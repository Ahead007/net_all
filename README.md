# net_all

### 在net user、net user add、net user active、net user del都禁用使用不了的场景下，利用windows用户操作api搞了个模拟这些功能的集合一起的工具。目前实现了如下功能：1、查看存在用户。2、查看指定用户信息。3、激活用户。4、添加用户。5、删除用户。测试通过最新版360以及火绒。

#### 使用方法：net_all.exe -h
